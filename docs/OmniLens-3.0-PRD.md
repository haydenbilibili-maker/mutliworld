## 1. 背景与现状（2.0 已达成）

OmniLens 已是一个**四空间（宇宙 / 近地 / 地表 / 洋底）**的全球态势实时可视化引擎，统一为「**空间(Tier) → 图层(Layer) → 视图(View)**」级联模型，数据**免密钥、真实、近实时、中立并陈**。

2.0 已落地能力：
- **近地**：风/洋流/海浪三套粒子流场（对标 earth.nullschool 技术，原创实现）；9+ 类标量叠加（污染物、颗粒物、海温、有效波高、海温偏差、度日热、白化预警、气温、降水、气压、湿度、CAPE）；数据栏含浓度色阶图例 + 悬停读数（流向/来源/时效）；视图菜单含投影、动画速度、配色方案。
- **地表**：USGS 实时地震（红波 + 海啸标记）、极光带、火山/风暴/洪水/沙尘/海冰、真彩卫星云图；实时态势计数条。
- **宇宙**：ISS/天宫实时星下点、NEO 接近事件面板、TLE 运行时自动获取（SGP4）。
- **洋底**：地震按震源深度、海缆/管线/板块。
- **工程基线**：强制防抖回归（0 ERROR）、按需懒加载、敏感图层一键下架、事件×行情关联引擎雏形。

**2.0 的结构性短板（3.0 要解决）**：
1. 投影仅 maplibre 平面/球面，无法对标竞品多投影。
2. 只有「此刻」——无时间轴/历史回放/预报推演。
3. 数据是「展示」而非「洞察」——缺少异常识别、关联推理、智能简报。
4. 无个人化（关注清单、阈值告警、分享、嵌入）。
5. 无协作与叙事（标注、快照、故事线）。
6. 未平台化（对外 API / 插件生态）。

---

## 2. 3.0 愿景与定位

> **从「看得见的实时地球」升级为「看得懂、可推演、能预警、可协作的决策级地球」。**

3.0 的一句话定位：**全球态势的「时间 × 空间 × 关联」决策引擎**——在四空间实时底座上，叠加**时间维度（回放/预报）**、**智能维度（异常/关联/简报）**与**人的维度（个人化/告警/协作/分享）**，并通过**开放平台**让能力外溢。

设计原则不变：真实数据非预测/示例/估算（预报须显式标注为预报）、免密钥优先、中立并陈、敏感可下架、原创复刻公开技术、强制防抖 0 ERROR。

---

## 3. 目标与北极星指标

**北极星指标**：周活跃「深度会话」数（单次会话发生 ≥1 次时间回放 / 关联钻取 / 生成简报 / 创建告警）。

| 维度 | 2.0 基线 | 3.0 目标 |
|---|---|---|
| 投影方案数 | 2（平面/球面） | ≥6（含等距方位/正射/温克尔三重/Waterman 等） |
| 时间能力 | 仅此刻 | 过去 7 天回放 + 未来 3–7 天预报 |
| 智能能力 | 无 | 异常榜 + 自动简报 + 自然语言问答 |
| 个人化 | 无 | 关注清单 + 阈值告警 + 分享链接/嵌入 |
| 对外 | 无 | 公开只读数据 API + 嵌入式 widget |
| 首屏 TTI | 待测 | ≤2.5s（4G） |
| 交互抖动 | 防抖 0 ERROR | 持续 0 ERROR，交互期 60fps 稳定 |

---

## 4. 用户与核心场景

- **研究/分析者**（地缘、能源、气候、海洋）：跨空间叠加 + 时间回放 + 关联钻取 + 导出。
- **媒体/科普/内容创作者**：多投影出图、快照、故事线、可嵌入 widget。
- **投资/政策关注者**：事件×行情关联、阈值告警、定制简报推送。
- **开发者/合作方**：调用只读数据 API、嵌入图层、贡献插件。
- **泛用户/教育**：好看好玩的活地球、按主题一键场景。

## 5. 能力规划（按主题模块）

> 每模块标注：目标 · 关键功能 · 验收 · 优先级（P0 必做 / P1 重要 / P2 增强）。

### A. 投影与渲染引擎升级 — P0
- **目标**：补齐对标 earth.nullschool 的多投影；渲染与 maplibre 解耦可插拔。
- **功能**：①新增「自绘渲染管线」（D3 投影 + WebGL/Canvas 叠加）作为近地/宇宙的可选渲染后端，提供 等距方位(A)、正射(O)、温克尔三重(W3)、Waterman 蝴蝶(WB)、立体(S)、等距柱(E) 等投影；②投影切换在「视图」菜单统一管理，平滑过渡动画；③粒子流场/标量叠加适配新投影坐标变换。
- **验收**：≥6 投影可切换且粒子/叠加正确贴合；切换 60fps 无白屏；防抖 0 ERROR。

### B. 时间机器（回放 + 预报） — P0
- **目标**：让态势从「此刻」扩展到「过去↔未来」。
- **功能**：①底部时间轴（已存在 TimelineSlider 升级）支持 过去 7 天 / 未来 3–7 天，步进播放/暂停/调速；②历史回放：地震/火山/风暴/洪水等事件按时间出现消失；气象/海洋标量场按时间帧切换（Open-Meteo 历史 + 预报；CRW/EONET 时间序列）；③**预报与实测显式区分**（预报帧加「预报」角标 + 虚线/降饱和），杜绝把预报当真值；④关键帧书签与对比（split/卷帘）。
- **验收**：时间轴拖动流畅、数据帧正确、预报标注清晰、移动端可用。

### C. 智能层（异常 · 关联 · 简报 · 问答） — P0/P1
- **目标**：从「展示数据」到「给出洞察」，承接 2.0 的关联引擎雏形。
- **功能**：①**异常榜**（P0）：跨层自动识别异常（强震 M≥6、BAA II 级、风暴逼近、污染超标、海温异常），按显著度排序，一键定位；②**自动态势简报**（P1）：按空间/区域/主题合成结构化中文摘要，标注来源与时效，可定时推送；③**事件×行情关联引擎成熟化**（P1）：灾害/地缘 ↔ 能源实物/股指/航运的关联卡片与时间对齐；④**自然语言问答**（P1）：对当前可见的真实数据做问答与导览（"最近 24h 哪里地震最强""把镜头带到台风路径"），仅基于已加载真实数据、不编造、给出处。
- **验收**：异常榜实时可点选；简报可读且不编造；问答答案均可溯源。

### D. 数据深度与广度 — P1
- **目标**：补齐高价值真实层并提升时效。
- **功能**：①新增真实层（候选）：海平面异常（卫星测高，需评估免密钥源）、闪电/雷暴、空气质量地面站（如可获免密钥源）、船舶/航班密度热力、野火烟羽、地磁/Kp 指数面板；②近地标量补充：云量、风暴潜势、海面盐度（视源）；③数据管线统一为「源适配器 + 缓存 + 时间索引」，支持时间机器；④数据健康看板（各源状态/时延/失败回退）。
- **验收**：新层免密钥真实、防抖 0 ERROR、健康看板可见。

### E. 个人化与告警 — P1
- **目标**：让用户「订阅」而非每次重看。
- **功能**：①关注清单（区域/图层/主题组合）一键恢复视图；②**阈值告警**（如"我关注区 M≥5 地震""台风进入 500km"）—— 站内 + 可选邮件/Webhook（账号与推送通道由用户自建/授权，平台不代建账户）；③视图偏好云端同步（在 2.0 localStorage 基础上）；④分享：生成带视图状态的深链 + 静态快照图 + 可嵌入 iframe widget。
- **验收**：关注清单可存取；告警按规则触发；分享链接可还原视图。
- **合规约束**：不代用户创建账户/输入凭据；告警通道需用户显式授权。

### F. 协作与叙事 — P2
- **目标**：把一次分析沉淀为可传播的故事。
- **功能**：①地图标注（点/线/圈 + 备注）；②快照集 → 故事线（多帧 + 文字解说，可播放）；③导出（PNG/MP4/GIF/PDF 简报）。
- **验收**：标注可存取分享；故事线可播放导出。

### G. 平台化（开放数据 + 插件） — P2
- **目标**：能力外溢，形成生态。
- **功能**：①公开**只读数据 API**（聚合后的标准化端点 + 速率限制 + 用量看板）；②**嵌入式 widget**（指定空间/图层/区域的可嵌 iframe/script）；③图层插件机制（第三方按「源适配器 + 渲染配置」贡献图层，沙箱化）。
- **验收**：API 文档 + 配额；widget 可嵌入第三方页；插件示例跑通。

### H. 性能 · 可用性 · 可达性 · 移动端 — P0（贯穿）
- **目标**：决策级稳定与广覆盖。
- **功能**：①**深度防抖 2.0**：消除所有交互期整图重绘，动画统一走 CSS/独立 canvas/节流；建立帧率监测与回归；②WebGL 自定义层承载海量点（卫星/船舶）；③移动端布局与触控手势完整适配；④可达性（键盘、对比度、reduced-motion 全覆盖）；⑤离线/弱网优雅降级 + 数据 stale 标注。
- **验收**：交互期 60fps；移动端可用；a11y 审计通过；防抖 0 ERROR。

### I. 商业化与合规 — P2
- **功能**：①免费（开放数据/科普）+ 专业版（告警/导出/API 配额/历史深度）分层；②中立性与免责声明体系（各国成果并陈、敏感层下架、预报免责）；③数据源许可合规审查（注明出处与使用条款）。

## 6. 信息架构与交互升级

- **三轴心智**：空间(Tier) × 时间(Timeline) × 图层/视图，左下「实时态势」、底部「时间机器」、右侧「洞察/简报/关联」、底栏「空间-图层-视图-时间」统一控制条。
- **场景预设**：一键进入「气候监测 / 灾害应急 / 地缘安全 / 海洋命脉 / 航天态势」等主题组合（自动选层+视图+时间窗）。
- **洞察优先**：异常榜与简报作为一级入口，从「找数据」变「数据找人」。

---

## 7. 技术架构演进

- **渲染**：maplibre（底图/球面）+ 新增「D3/WebGL 自绘叠加层」可插拔后端；统一坐标变换抽象，图层声明渲染后端。
- **数据**：统一「源适配器（fetch+解析+缓存+时间索引）」；服务端聚合 + Edge 缓存；时间机器走帧索引；数据健康看板。
- **智能**：异常规则引擎（声明式阈值）+ 关联规则 + LLM 简报/问答（仅基于已取真实数据，强约束不编造、强制出处）。
- **状态**：Zustand 视图状态 + 云端偏好/关注/告警（需后端持久层，评估方案）。
- **质量**：防抖回归扩展为「0 ERROR + 帧率门禁」；端到端冒烟；类型完整性 CI。

---

## 8. 非功能需求

- 真实数据原则（预报显式标注）；免密钥优先；中立并陈；敏感一键下架。
- 性能：首屏 TTI ≤2.5s（4G）；交互 60fps；首屏 JS 持续瘦身（懒加载）。
- 可达性：键盘可达、对比度达标、reduced-motion 全覆盖。
- 隐私/合规：不代建账户/不代输凭据；告警通道用户授权；数据源出处与条款标注。
- 稳定：所有动画零交互期整图重绘；失败优雅降级 + stale 标注。

---

## 9. 里程碑与排期（建议）

- **M1（投影 + 时间机器底座）**：D3/WebGL 自绘后端 + ≥4 投影；时间轴回放（事件层）+ 预报标注。→ 价值：补齐对标 + 时间维度。
- **M2（智能层）**：异常榜 + 自动简报 + 关联卡片成熟化。→ 价值：从展示到洞察。
- **M3（个人化 + 分享）**：关注清单 + 阈值告警 + 深链/快照/嵌入 widget。→ 价值：留存与传播。
- **M4（平台化 + 协作 + 商业化）**：只读 API + 插件 + 故事线 + 分层方案。→ 价值：生态与变现。

（每个里程碑结束须：防抖 0 ERROR、帧率门禁通过、关键路径冒烟、文档更新。）

## 10. 风险与依赖

| 风险 | 影响 | 缓解 |
|---|---|---|
| D3 多投影 = 重渲染重写 | 高（M1 核心） | 作为可插拔后端、与 maplibre 并存、分投影灰度 |
| 部分高价值源需 key/付费（OpenAQ v3、卫星测高） | 中 | 优先免密钥源；需 key 的明确标注或降级 |
| 时间机器历史数据量与缓存成本 | 中 | 帧索引 + 粗网格 + Edge 缓存 + 限窗 |
| LLM 简报/问答编造风险 | 高（信任） | 仅基于已取真实数据、强制出处、不可溯源不输出 |
| 告警/账号合规 | 中 | 不代建账户、用户授权通道、明确免责 |
| 抖动回归 | 中 | 防抖门禁 + 帧率监测纳入 CI |

## 11. 度量与验收

- 北极星：周深度会话数；次级：回放使用率、简报打开率、告警创建/触发数、分享/嵌入数、API 调用数。
- 工程：防抖 0 ERROR、交互 60fps、首屏 TTI、错误率、数据源可用率。

---

## 12. 非目标（Out of Scope）

- 不做真实交易/下单/资金操作；不提供个性化投资建议（仅陈列事实）。
- 不代用户创建账户、不代输入任何凭据。
- 不输出无法溯源的"数据"；预报一律显式标注，绝不当作实测。
- 不偏袒任一国家立场（中立并陈各国成果）。
